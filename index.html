Contexte du projet

Vous faites partie d’une équipe de développement web dans une agence spécialisée dans la création de sites web. L’équipe utilise Git pour le contrôle de version et GitHub pour héberger les projets. Vous devez initialiser un projet de site web statique sur GitHub, gérer les branches de développement, et collaborer avec les autres membres de l’équipe en respectant les bonnes pratiques de Git.
